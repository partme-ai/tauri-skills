# CLI Reference

**官方文档**: https://v2.tauri.org.cn/reference/cli/

## Common Commands

- tauri init
- tauri dev
- tauri build
- tauri info
- tauri icon

### Example: Init

```bash
cargo tauri init
```

### Example: Dev

```bash
cargo tauri dev
```

### Example: Build

```bash
cargo tauri build
```

### Key Points

- Use cargo for Rust-first workflows
- Use npm/yarn/pnpm for JS-first workflows
